Wat gebeurt er als eenzelfde object meermaals voorkomt bij het opbouwen van de JSON tekst?

Wanneer eenzelfde object meerdere keren wordt gebruikt bij het genereren van een JSON-tekst, zal het object volledig worden opgenomen in de tekst elke keer dat het voorkomt, wat leidt tot onnodige redundantie en een grotere JSON-tekst.
